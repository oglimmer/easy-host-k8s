Overview

easy-host lets you upload HTML files or ZIP archives and serve them on unique slug-based URLs. Upload via the REST API or the web UI, and your content is instantly available at /s/{slug}.

API Reference

The full interactive API documentation is available via Swagger UI.

Open Swagger UI OpenAPI JSON

Authentication

The REST API uses HTTP Basic Auth. Include your credentials with every request.

curl -u username:password https://content.oglimmer.com/api/content

Quick Start

1. Upload a single HTML file

curl -u username:password \
  -F "slug=my-page" \
  -F "file=@index.html" \
  -F "title=My Page" \
  https://content.oglimmer.com/api/content

2. Upload a ZIP archive

curl -u username:password \
  -F "slug=my-site" \
  -F "file=@site.zip" \
  -F "title=My Site" \
  https://content.oglimmer.com/api/content

3. Access your content

https://content.oglimmer.com/s/my-page

Slug Rules

Slugs must match the pattern ^[a-z0-9][a-z0-9-]*[a-z0-9]$:

  • Lowercase letters, digits, and hyphens only
  • Must start and end with a letter or digit
  • Minimum 2 characters

Rate Limiting

API requests are limited to 10 requests per second per IP address. Exceeding this limit returns 429 Too Many Requests.

Upload Limits

Maximum file size: 10 MB. ZIP archives are extracted server-side; macOS metadata directories (__MACOSX) and hidden files are automatically filtered out.
