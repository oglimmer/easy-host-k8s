Authentication

The API uses HTTP Basic Authentication. Include your credentials in the Authorization header.

Quick Start

# Upload content
curl -u admin:changeme -F "slug=hello" -F "file=@index.html" \
  http://localhost:8080/api/content

# View it
open http://localhost:8080/s/hello

API Endpoints

MethodPathDescription
GET/api/contentList all your content
POST/api/contentCreate new content (multipart: slug, file, title?, sourceUrl?, creator?)
GET/api/content/{slug}Get content details
PUT/api/content/{slug}Update content
DELETE/api/content/{slug}Delete content

Slug Rules

  • Lowercase letters, numbers, and hyphens only
  • Must start and end with a letter or number
  • Pattern: ^[a-z0-9]([a-z0-9-]*[a-z0-9])?$

Limits

  • Max upload size: 10 MB
  • Rate limit: 10 requests/second per IP
  • ZIP uploads: __MACOSX and hidden files are filtered
